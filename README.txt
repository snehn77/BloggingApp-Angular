For running Backend - 

1) Run command - npm start to start the backend 
2) Once you see the message running on port and database connect than means backend server is running

To run the front end- 

1) Go on other terminal 
2) chage directory and go inside the client folder using cd client
3) Run npm start to start the front - end server
4) Wait it will automatically load once build is complete


Software Used - 
1) Visual Studio Code
2) MangoDB (For database)
